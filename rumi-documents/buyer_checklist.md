# 🛒 Rumi Buyer’s Checklist (Peru)

## Step 1: Provenance Verification
- [ ] **Concession ID (INGEMMET)** → Ask the store for the mining concession number and cross‑check in the INGEMMET registry: https://www.ingemmet.gob.pe
- [ ] **REINFO ID (MINEM)** → Confirm the miner is registered in REINFO: https://www.gob.pe/minem
- [ ] **Vendor RUC (SUNAT)** → Request the store’s tax ID and verify in the SUNAT RUC database: https://www.sunat.gob.pe
- [ ] **HS Code (MINCETUR)** → Ensure the stone is classified correctly for customs; confirm via MINCETUR trade database: https://www.mincetur.gob.pe
- [ ] **VUCE COD** → If the stone is for export, ask for the Certificate of Origin code and verify through VUCE: https://www.vuce.gob.pe

## Step 2: Artisan Verification (Jewelry Only)
- [ ] **Artisan RNA** → Mandatory for jewelry; confirm the artisan is registered in the National Artisan Registry: https://www.gob.pe/mincetur
- [ ] **Mount Type** → Ask how the stone is mounted (bezel, prong, filigree).
- [ ] **Artisan Technique** → Optional, but valuable for cultural storytelling.
- [ ] **Artisan Signature** → Optional digital signature/hash for authenticity.

## Step 3: Compliance Proof
- [ ] **Hedera Consensus Service (HCS)** → Ensure the compliance payload has been notarized. Explorer: https://hashscan.io
- [ ] **Transaction ID** → Verify the `compliance_proof_hcs` is attached to metadata and matches the NFT pinned to IPFS.
- [ ] **IPFS CID** → Confirm the CID is accessible and immutable via IPFS gateway: https://ipfs.io

## Exporter Companies & Sources in Peru (Reference)
- **Peruvian Minerals & Jewelry S.A.C.**
  - Focus: Semi‑precious stones (chrysocolla, pyrite, cabochons, spheres, jewelry).
  - Provides: Export documentation, HS Codes, RUC, provenance data.
  - Value: Suitable for Rumi artisanal certification.

- **Inka Minerals Peru**
  - Focus: Wholesale minerals (chrysocolla, pyrite, galena, bolivianite).
  - Provides: HS Codes, export records, provenance documentation.
  - Value: Operates at export scale, compliant with customs.

## National Registries
- **PROMPERÚ Export Registry (Exportemos Portal)** → https://www.exportemos.pe
- **MINCETUR Trade Database** → https://www.mincetur.gob.pe

## Industry Associations
- **ADEX (Asociación de Exportadores)** → https://www.adexperu.org.pe

# 📌 Buyer’s Action Plan
1. Source directly from exporters in Lima or those listed in PROMPERÚ/MINCETUR.
2. Request and cross‑check documentation:
   - Concession ID (INGEMMET registry: https://www.ingemmet.gob.pe)
   - REINFO ID (MINEM registry: https://www.gob.pe/minem)
   - Vendor RUC (SUNAT database: https://www.sunat.gob.pe)
   - HS Code (MINCETUR trade database: https://www.mincetur.gob.pe)
   - VUCE COD (certificate of origin: https://www.vuce.gob.pe)
3. For jewelry → also request artisan RNA and keep a copy.
4. Verify compliance proof:
   - Confirm HCS transaction ID (`compliance_proof_hcs`) matches NFT metadata (https://hashscan.io).
   - Check IPFS CID is pinned and accessible (https://ipfs.io).
   - Cross‑check on Hedera Explorer before purchase.
